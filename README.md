# FB Movie Night (Work In Progress)

I created this project to learn React. It is using React, Firebase, and Firebase Authentication using Slack's Custom Token.

## Getting Started

### Installing

```
npm install
```

To Run

```
npm run dev
```

Checkout the project on

```
http://localhost:3333/
```
